# 🛒 Python Shop Parser

Парсер товаров с сайта интернет-магазина с пагинацией, сортировкой и сохранением данных.

---

## 📌 Описание

Проект собирает данные о товарах с сайта:

* название товара
* цена
* ссылка на товар
* изображение

Дополнительно:

* переход по страницам (pagination)
* сортировка товаров по цене
* сохранение данных в CSV и JSON

---

## ⚙️ Технологии

* Python 3
* Requests
* BeautifulSoup4
* CSV
* JSON
* urllib

---

## 🚀 Возможности

* Парсинг товаров с нескольких страниц
* Обработка HTML-страниц
* Сбор структурированных данных
* Сортировка товаров по цене (по убыванию)
* Экспорт данных в:

  * `shop.csv`
  * `shop.json`
